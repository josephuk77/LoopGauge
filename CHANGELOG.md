# Changelog

All notable changes are documented here. LoopGauge follows semantic versioning after the experimental `0.x` line.

## [0.2.0] - 2026-07-16

- Add an API-free synthetic replay demo.
- Add reproducible benchmark methodology and contribution templates.
- Add GitHub Actions CI for Node.js 22 and 24.

## [0.1.0] - 2026-07-16

- Initial provider-neutral CLI and MCP server.
- Add OpenAI Codex and Anthropic adapters.
- Add isolated Git-worktree optimization, quality gates, cost accounting, SQLite state, and JSONL traces.
- Add same-provider automatic model discovery from a dated catalog and provider Models APIs.
