# 행동강령

[English](CODE_OF_CONDUCT.md) | 한국어

LoopGauge는 측정 근거를 우선하는 엔지니어링 프로젝트입니다. 모든 참여자는 서로를 존중하고, 구체적으로 소통하며, 안전을 지켜야 합니다.

## 기대하는 행동

- 사람을 공격하지 않고 아이디어와 결과를 논의합니다.
- 측정된 근거와 시뮬레이션, 추정, 의견을 명확히 구분합니다.
- 개인정보, 라이선스, 공급자 약관과 고객 기밀을 존중합니다.
- 벤치마크, grader, 데이터셋과 선행 작업의 출처를 밝힙니다.
- 처음 참여하는 사람이 최소한의 재현 가능한 사례를 만들도록 돕습니다.

## 허용하지 않는 행동

괴롭힘, 차별, 협박, 신상 공개, 비밀정보 노출, 조작된 벤치마크 주장, 비공개 프로젝트 데이터 공개는 허용되지 않습니다.

관리자는 커뮤니티 보호를 위해 필요한 경우 콘텐츠를 수정·삭제하거나 참여를 제한할 수 있습니다. 민감한 행동 또는 보안 문제는 공개 이슈가 아닌 GitHub 비공개 채널로 신고해 주세요.
