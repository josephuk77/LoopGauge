# 변경 이력

[English](CHANGELOG.md) | 한국어

주요 변경 사항을 이 문서에 기록합니다. LoopGauge는 실험 단계인 `0.x` 버전 이후부터 유의적 버전 규칙을 따릅니다.

## [0.2.0] - 2026-07-16

- API 호출이 필요 없는 합성 재현 데모를 추가했습니다.
- 재현 가능한 벤치마크 방법론과 기여 양식을 추가했습니다.
- Node.js 22와 24용 GitHub Actions CI를 추가했습니다.
- 모든 사용자·기여자 문서에 한국어판을 추가했습니다.

## [0.1.0] - 2026-07-16

- 공급자 중립형 CLI와 MCP 서버의 초기 버전을 추가했습니다.
- OpenAI Codex와 Anthropic 어댑터를 추가했습니다.
- 격리된 Git worktree 최적화, 품질 하한, 비용 계산, SQLite 상태 저장과 JSONL 추적을 추가했습니다.
- 날짜가 기록된 카탈로그와 공급자 Models API에서 동일 공급자의 저비용 모델을 자동 탐색하는 기능을 추가했습니다.
