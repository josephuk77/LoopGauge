# API 호출 없는 합성 재현

[English](README.md) | 한국어

이 fixture는 품질 하한에 미달하는 저비용 정책을 LoopGauge가 제외하고, 승인 작업당 비용을 기준으로 검증 절차가 포함된 정책을 선택하는 과정을 보여줍니다.

공급자 API를 한 번도 호출하지 않습니다. 이름, 점수, 비용, 절감률과 손익분기 값은 모두 합성 데이터입니다. 실제 모델 측정 결과로 인용하면 안 됩니다.

```bash
npm run demo
```

예상 결정은 다음과 같습니다.

- `haiku-direct`는 가격이 낮지만 품질 하한에 미달하므로 제외합니다.
- `sonnet-verify`와 `haiku-guarded`는 95% 품질 하한을 통과합니다.
- 합성 승인 작업당 비용이 더 낮은 `haiku-guarded`를 선택합니다.
- 최적화 비용을 포함한 합성 손익분기점을 보고합니다.
