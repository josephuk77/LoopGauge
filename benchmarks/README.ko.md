# LoopGauge 벤치마크

[English](README.md) | 한국어

벤치마크는 한 가지 주장을 검증합니다. 더 저렴한 정책이 정의된 품질 하한을 만족하면서 성공 작업당 총비용을 낮출 수 있는가?

현재 저장소에는 온보딩을 위한 API 호출 없는 합성 재현이 포함되어 있습니다. 이 데모는 리포트의 의미만 보여주며 실제 모델 성능이나 비용 절감의 근거가 아닙니다.

```bash
npm install
npm run demo
npm run demo -- --json
```

실제 벤치마크를 기여할 때는 [벤치마크 방법론](../docs/benchmark-methodology.ko.md)을 따라야 하며, 라이선스와 개인정보 보호가 허용하는 범위에서 비식별화된 원시 관측값을 공개해야 합니다.
