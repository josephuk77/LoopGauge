# Code of Conduct

LoopGauge is an evidence-first engineering project. Participation should be respectful, specific, and safe.

## Expected behavior

- Discuss ideas and results without attacking people.
- Separate measured evidence from simulations, estimates, and opinions.
- Respect privacy, licensing, provider terms, and customer confidentiality.
- Give credit for benchmarks, graders, datasets, and prior work.
- Help newcomers produce minimal, reproducible examples.

## Unacceptable behavior

Harassment, discrimination, threats, doxxing, secret exposure, fabricated benchmark claims, and publishing private project data are not acceptable.

Maintainers may edit or remove content and restrict participation when necessary to protect the community. Report sensitive conduct or security concerns through private GitHub channels rather than public issues.
